Welcome to my repository! This project contains my personal solutions and progress tracking for the highly curated [**Curious Freaks Coding Sheet** by Nandhini](https://docs.google.com/spreadsheets/d/1P3RXgZju_2OzZyJaRtu6D3Kx-Eks38X09_zJ8aj2zy8/).

This repository currently encompasses comprehensive solutions for problems ranging from foundational concepts up to the **Sliding Window Technique** (Rows 18 to 233 of the sheet).

## 📚 Topics Covered So Far

The solutions are cleanly organized into packages covering the following core algorithmic paradigms:

* **🧮 Basic Math:** Fundamental mathematical operations and logic building.
* **🎨 Pattern:** Loops and logic building through visual pattern problems.
* **➕ Prefix Sum:** Optimizing range queries and cumulative sums.
* **🧩 Sub-Array:** Contiguous sequence manipulation and optimization.
* **🔲 Matrix:** 2D array traversal, searches, and transformations.
* **🔤 String:** Character manipulation, pattern matching, and parsing.
* **🔄 Recursion:** Problem-solving using self-referential function calls.
* **🔍 Binary Search:** Efficient searching algorithms and variations on sorted datasets.
* **📊 Sorting:** Algorithmic techniques for ordering data efficiently.
* **✌️ Two Pointers:** Optimized array and string traversal techniques.
* **🪟 Sliding Window:** Fixed and variable-size window techniques for subarray/substring problems.

## 🚀 How to Run Locally

If you'd like to run these solutions on your local machine:

1. Clone the repository:
   ```bash
   git clone [https://github.com/YourUsername/YourRepositoryName.git](https://github.com/YourUsername/YourRepositoryName.git)
   ```
2. Open the project in IntelliJ IDEA (or your preferred Java IDE).
3. Navigate to the specific topic folder, open the Java file, and run the ```main``` method.
